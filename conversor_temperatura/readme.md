# Verificador de Temperatura

Programa em Python que permite ao usuário digitar a temperatura e receber recomendações sobre o clima.

**Curso:** Programação em Python – SENAC  
**Arquivo:** `temperatura.py`

## Funcionalidades

1. Aceita várias entradas consecutivas sem precisar reiniciar.  
2. Detecta automaticamente se a temperatura foi digitada em Celsius, Kelvin ou Fahrenheit e converte para Celsius.  
3. Recomenda ações de acordo com a temperatura: agasalhar, se hidratar, etc.  
4. Trata entradas inválidas sem quebrar o programa.  
5. Permite encerrar digitando `"sair"`.

1. Coloque o arquivo `temperatura.py` na pasta do repositório.  
2. Abra o terminal nessa pasta.

4. Digite a temperatura ou `"sair"` para encerrar.
